# Carl-Jake-GroceryStore

This is a grocery store android app. Simply a base template android app for any store. 
Due to its clean code style this app is easily modifiable.


Features of this app:

	1.) Welcome Screen
    2.) Home Screen (lists random products)
    3.) Categorized Product Tabs
    3.) Product Description Tab
    4.) A Shopping Cart with Check Out
    5.) Item Location In Store (Very Basic)
    4.) 2 Databases (1 for checkout and 1 for product inventory)
